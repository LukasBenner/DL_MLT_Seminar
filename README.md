# Deep transformation models: Tackling complex regression problems with neural network based transformation models

Dieses Repository beschäftigt sich mit Deep Transformation Models, einer Methode zur Lösung komplexer Regressionsprobleme mithilfe neuronaler Netzwerke und Transformationen. Es basiert auf dem Paper *[Deep transformation models: Tackling complex regression problems with neural network based transformation models](https://ieeexplore.ieee.org/abstract/document/9413177)*
